# Коммит изменений

Команда для создания коммита с изменениями.

## Использование

```bash
git add .
git commit -m "описание изменений"
git push
```

## Варианты

- Коммит всех изменений: `git add . && git commit -m "сообщение" && git push`
- Коммит конкретных файлов: `git add file1 file2 && git commit -m "сообщение" && git push`
- Коммит без push: `git add . && git commit -m "сообщение"`
